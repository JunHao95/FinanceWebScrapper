Target shape after this plan:
```python
# New reads
page_context = data.get("context", "")   # plain text string or ""
history = data.get("history", [])         # [{sender: "user"|"bot", text: "..."}]

# Context injection
effective_system_prompt = system_prompt
if page_context and page_context.strip():
    effective_system_prompt = system_prompt + "\n\n" + page_context.strip()

# History mapping helper (inline or small helper)
def _map_role(sender):
    return "assistant" if sender == "bot" else "user"

history_messages = [
    {"role": _map_role(h.get("sender", "user")), "content": h.get("text", "")}
    for h in history[-10:]   # safety cap at 10
    if h.get("text")
]

# Extended Groq payload
payload = {
    "model": MODEL,
    "messages": [
        {"role": "system", "content": effective_system_prompt},
        *history_messages,
        {"role": "user", "content": message}
    ],
    "max_tokens": 512,
    "temperature": 0.7
}
```

Same extended messages structure must be applied to the Ollama branch payload too (Ollama uses OpenAI-compatible format).
