**480px block** (phone — the main responsive fix):
```css
/* ── 480px: phone ── */
@media (max-width: 480px) {
    body { padding: 10px; }
    header { padding: 16px; }
    header h1 { font-size: 1.4rem; }
    .input-section { padding: 16px; }

    /* Inner results tab bar: switch from wrap to horizontal scroll */
    .tabs {
        flex-wrap: nowrap;
        overflow-x: auto;
        -webkit-overflow-scrolling: touch;
        scrollbar-width: none;        /* Firefox */
    }
    .tabs::-webkit-scrollbar { display: none; }  /* WebKit */
    .tab-button {
        padding: 10px 16px;
        font-size: 14px;
        white-space: nowrap;
        flex-shrink: 0;
    }

    /* Main tab bar: scrollbar hiding (overflow-x already set at line 1045) */
    .main-tab-buttons {
        scrollbar-width: none;
    }
    .main-tab-buttons::-webkit-scrollbar { display: none; }
    .main-tab-button { white-space: nowrap; flex-shrink: 0; }

    /* Chatbot mobile sizing */
    #chatbot-toggle-btn {
        width: 44px;
        height: 44px;
        font-size: 20px;
    }
    #chatbot-window {
        width: calc(100vw - 20px);
        right: 10px;
    }

    /* Allocation rows: single column */
    .allocation-input-row {
        flex-direction: column;
        align-items: stretch;
    }
    .allocation-input-row label { flex: none; }
    .allocation-input-row input { width: 100%; }

    /* Single-column grids */
    .analysis-grid { grid-template-columns: 1fr; }
    .option-results-grid { grid-template-columns: 1fr; }
    /* .metrics-grid already 1fr from 768px block — no duplicate needed */

    /* Table overflow */
    .table-scroll-wrap {
        overflow-x: auto;
        -webkit-overflow-scrolling: touch;
    }
}
```
