**Test:** Click the "FinancialAnalyst" pill.
**Expected:** "FinancialAnalyst" pill gains `.agent-pill.active` styling (white border, full opacity); "QuantAssistant" becomes dim.
**Why human:** CSS class toggling and visual state change cannot be verified statically.

### 3. Header Title and Subtitle Update

**Test:** With the chat open, click "FinancialAnalyst" pill.
**Expected:** The h3 in the header updates to "📈 FinancialAnalyst" and the subtitle below it changes to "Company fundamentals & market analysis".
**Why human:** Runtime DOM mutation via `updateHeader()`; requires live browser session.

### 4. Agent History Swap

**Test:** Send a message as QuantAssistant, then click "FinancialAnalyst". Then click "QuantAssistant" again.
**Expected:** After switching to FinancialAnalyst, only the FinancialAnalyst greeting appears (no QuantAssistant messages). After switching back, the QuantAssistant conversation (your message + its reply) is fully restored.
**Why human:** In-memory `agentHistories` swap; requires a live browser session with multiple messages.

### 5. Widget Open/Close History Persistence

**Test:** Send messages in both agent threads. Click the × button to close the widget. Click the chat bubble to reopen it.
**Expected:** Both agent threads are intact (toggling between pills shows each thread's messages).
**Why human:** Persistence of in-memory JS objects across widget hide/show cycles requires a browser session to confirm.
