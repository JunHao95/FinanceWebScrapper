R_mat = np.array([
    [ 2.0,  0.0, -2.0],   # risk_off: underweight rewarded, overweight penalized

Default transition matrices (one per action — action-dependent dynamics):
```python
# P[action][state, next_state]
P_default = np.array([
    # underweight action: risk_off is sticky, slowly recovers
    [[0.7, 0.2, 0.1], [0.3, 0.5, 0.2], [0.1, 0.3, 0.6]],
    # neutral action: moderate mean-reversion
    [[0.6, 0.3, 0.1], [0.2, 0.6, 0.2], [0.1, 0.3, 0.6]],
    # overweight action: risk_on is sticky
    [[0.6, 0.2, 0.2], [0.2, 0.4, 0.4], [0.1, 0.2, 0.7]],
])
```

If `transition_override` is provided (list-of-lists-of-lists, 3x3x3), convert to numpy and use instead of P_default.

Value iteration loop (verified in research at 371 iterations for gamma=0.95, tol=1e-8):
```python
V = np.zeros(3)
converged = False
for iteration in range(n_periods):
    V_old = V.copy()
    Q_vals = np.zeros((3, 3))  # state x action Q-values
    for a in range(3):
        Q_vals[:, a] = R_mat[:, a] + gamma * P[a] @ V
    V = Q_vals.max(axis=1)
    policy = Q_vals.argmax(axis=1)
    if np.max(np.abs(V - V_old)) < tol:
        converged = True
        break
```

Return:
```python
{
    'optimal_policy': policy.tolist(),          # [int, int, int] — action index per state
    'value_function': V.tolist(),               # [float, float, float]
    'convergence_iterations': iteration + 1,
    'converged': converged,
    'states': ['risk_off', 'neutral', 'risk_on'],
    'actions': ['underweight', 'neutral', 'overweight'],
    'gamma': gamma,
}
```
