Create static/js/dcfValuation.js as an IIFE module following earningsQuality.js structure exactly. Key implementation details:

**parseNumeric** — copy from earningsQuality.js but ADD comma stripping: `val = val.replace(/,/g, '')` before the `parseFloat` call. This is critical for Market Cap (Yahoo) and Free Cash Flow (Yahoo) which use comma-formatted strings (e.g., "12,345,678").

**extractMetric** — copy verbatim from earningsQuality.js (alias-based lookup, case-insensitive substring match, null-safe).

**Module-level data cache** — `const _dataCache = {};` inside the IIFE, keyed by ticker. `renderIntoGroup` writes `_dataCache[ticker] = data` so `_recalculate` can access the original data without DOM attributes or pageContext dependency.

**computeValuation(data, wacc, g1, g2)**:
1. Extract FCF: `avFcf = extractMetric(data, ['Free Cash Flow (AlphaVantage)'])`. Extract `yfFcf = extractMetric(data, ['Free Cash Flow (Yahoo)'])`. Use `fcf0 = (avFcf !== null && avFcf !== 0) ? avFcf : yfFcf`. Set `fcfSource` accordingly.
2. Guard: if `fcf0 === null || fcf0 === 0`, return `{ error: 'FCF data missing', fcfSource: null }`.
3. Guard: if `wacc <= g2`, return `{ error: 'WACC must exceed terminal growth rate', fcfSource }`.
4. Stage 1: loop t=1..5, `fcft = fcf0 * Math.pow(1 + g1, t)`, accumulate `stage1PV += fcft / Math.pow(1 + wacc, t)`. Capture `fcf5` at t=5.
5. Stage 2: `terminal = fcf5 * (1 + g2) / (wacc - g2)`. `pvTerminal = terminal / Math.pow(1 + wacc, 5)`. `intrinsicEquityTotal = stage1PV + pvTerminal`.
6. Shares: `marketCap = extractMetric(data, ['Market Cap (Yahoo)'])`. `currentPrice = extractMetric(data, ['Current Price (Yahoo)', 'Current Price'])`. If both non-null and currentPrice != 0: `sharesOut = marketCap / currentPrice`, `intrinsicPerShare = intrinsicEquityTotal / sharesOut`, `premium = (currentPrice - intrinsicPerShare) / intrinsicPerShare * 100`.
7. Return `{ intrinsicEquityTotal, intrinsicPerShare, premium, fcfSource }`.

**buildHTML(ticker, result, defaultWacc, defaultG1, defaultG2)**:
- If `result.error === 'FCF data missing'`: return a single row `<div class="metric-item"><span class="badge badge-warning">DCF unavailable — FCF data missing</span></div>`.
- If `result.error` (other, e.g. WACC guard): show inline error message.
- Otherwise render:
  - Collapsed header (emoji-header pattern): `<span>💰 DCF Value: $XX.XX  ▼</span>` — use `intrinsicPerShare` if available, else `($XX.XXB equity)`.
  - Expanded content: intrinsic per-share (or equity total in $B), premium/discount badge (`.badge-success` for discount / `.badge-danger` for premium), FCF source footnote, assumption inputs (WACC, g1, g2 as `<input type="number">` with default values in percentage form: 10, 10, 3), Recalculate button.
  - Input IDs: `dcf-wacc-{ticker}`, `dcf-g1-{ticker}`, `dcf-g2-{ticker}`. Result span ID: `dcf-result-{ticker}`. Premium span ID: `dcf-premium-{ticker}`.
  - Recalculate button: `onclick="DCFValuation._recalculate('${ticker}')"`.
  - Follow collapsed/expanded toggle pattern from healthScore.js — use a toggle div with onclick.

**renderIntoGroup(ticker, data, cardRoot)**:
- Query `#deep-analysis-content-{ticker}`; return silently if not found.
- Store `_dataCache[ticker] = data`.
- Call `computeValuation(data, 0.10, 0.10, 0.03)`.
- Build HTML via `buildHTML`, append a `<div class="dcf-section">` to the container.

**_recalculate(ticker)**:
- Read inputs: `wacc = parseFloat(document.getElementById('dcf-wacc-' + ticker).value) / 100`.
- `g1 = parseFloat(document.getElementById('dcf-g1-' + ticker).value) / 100`.
- `g2 = parseFloat(document.getElementById('dcf-g2-' + ticker).value) / 100`.
- Retrieve `data = _dataCache[ticker]`.
- Call `computeValuation(data, wacc, g1, g2)`.
- Update `#dcf-result-{ticker}` and `#dcf-premium-{ticker}` in-place (do NOT replace whole section).
- Expose as `window.DCFValuation._recalculate = _recalculate` so inline onclick can call it.

**clearSession()** — no-op, mirrors Phase 13/14 pattern.

**Exports**: `window.DCFValuation = { computeValuation, renderIntoGroup, clearSession }`. Also add `module.exports` guard for Node.js smoke test.
