       - `ed = yf.Ticker(ticker).earnings_dates`
       - Guard: if None, empty, or 'Reported EPS' not in columns -> return None
       - Filter: `past = ed[ed['Reported EPS'].notna()]`; if empty -> return None
       - Strip tz: `last = past.index.max(); return last.tz_localize(None) if last.tzinfo is not None else last`
       - Wrap entire body in `try/except Exception: return None`

    2. Internal helper `_avwap_series(df_full, anchor_date, display_index) -> pd.Series`:
       - `subset = df_full.loc[anchor_date:]`; if empty -> return pd.Series(index=display_index, dtype=float)
       - `tp = (subset['High'] + subset['Low'] + subset['Close']) / 3.0`
       - `cum_v = subset['Volume'].cumsum().replace(0, np.nan)` (guard zero-volume)
       - `cum_tpv = (tp * subset['Volume']).cumsum()`
       - `return (cum_tpv / cum_v).reindex(display_index)` — NaN before anchor handled by connectgaps

    3. Internal helper `_safe_list(series) -> list`:
       - `return [None if pd.isna(v) else float(v) for v in series]`

    4. Main `compute_anchored_vwap(df: pd.DataFrame, ticker: str, lookback: int) -> dict`:
       - `df` is already the 365-day OHLCV (caller provides)

       - Convergence check: `abs(current_price - val) / current_price <= 0.003` for each line
       - Signal: 'above' if current_price > avwap_high_val (if not NaN), 'below' if current_price < avwap_low_val, else 'between'
       - Labels: `f'52-wk High: {pct:+.1f}%'` where `pct = (current_price - avwap_high_val) / avwap_high_val * 100`
