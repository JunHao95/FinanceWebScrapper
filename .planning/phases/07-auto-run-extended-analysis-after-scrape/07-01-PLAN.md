triggerAutoRun(tickers):
  1. Compute dates:
     const today = new Date();
     const endDate   = formatDate(today);
     const d2y = new Date(today); d2y.setFullYear(d2y.getFullYear() - 2);
     const startDate = formatDate(d2y);
     const d7y = new Date(today); d7y.setFullYear(d7y.getFullYear() - 7);
     const mdpStart  = formatDate(d7y);
     const trainEnd  = startDate;   // same as regime startDate (2y ago)
     const testStart = startDate;
  2. Get #analyticsResults element. If missing, return early (no-op).
  3. Remove any existing #autoRunSection (so re-runs don't duplicate): document.getElementById('autoRunSection')?.remove()
  4. Inject HTML: document.getElementById('analyticsResults').insertAdjacentHTML('afterbegin', buildAutoRunHTML(tickers))
  5. Build promises array:
     const regimePromises = tickers.map(t => runAutoRegime(t, startDate, endDate));
     const allPromises = tickers.length >= 2
       ? [...regimePromises, runAutoMDP(mdpStart, trainEnd, testStart)]
       : regimePromises;
  6. await Promise.allSettled(allPromises) — do nothing with results (each sub-function handles its own UI)

runAutoRegime(ticker, startDate, endDate):
  async function, catches all errors internally (never throws).
  1. Fetch POST /api/regime_detection with JSON { ticker, start_date: startDate, end_date: endDate }
  2. If fetch throws or resp.ok === false or data.success === false:
     - Set badge: document.getElementById('autoRegimeBadge_' + ticker).setAttribute('style', BADGE_FAILED); badge.textContent = 'Failed'
     - Set placeholder: document.getElementById('autoRegimePlaceholder_' + ticker).textContent = 'Regime detection failed: ' + (err?.message || data?.error || 'Unknown error')
     - return
  3. On success:
     - Hide placeholder: document.getElementById('autoRegimePlaceholder_' + ticker).style.display = 'none'
     - Show chart divs: set display:block on autoRegimeProb_{ticker} and autoRegimePrice_{ticker}
     - Call renderRegimeCharts(ticker, data)
     - Update badge: style = BADGE_DONE, textContent = 'Done'

runAutoMDP(mdpStart, trainEnd, testStart):
  async function, catches all errors internally.
  1. Fetch POST /api/stoch_portfolio_mdp with JSON:

     - Set badge: document.getElementById('autoMDPBadge').setAttribute('style', BADGE_FAILED); badge.textContent = 'Failed'
     - document.getElementById('autoMDPResults').innerHTML = rlAlert('Portfolio MDP failed: ' + err.message)
     - return
  3. On success:
     Destructure same fields as rlModels.js runPortfolioMDP() (lines 367-371):
       { equity_ticker, bond_ticker, optimal_policy, state_names, action_names, v_star, q_matrix, state_counts, iterations, test_dates, rl_cumret, benchmark_cumret, perf_metrics }

     Update badge: style = BADGE_DONE, textContent = 'Done'
