**Step 2: Markov chain result — add transition matrix heatmap.**
In the existing Markov results function (likely `runMarkovChain` or similar), after the current response handling, add Plotly chart rendering:

For `credit_transitions` mode (or the default steady-state mode where a transition matrix is returned):
```javascript
// After existing result HTML injection, add chart container and render heatmap
if (data.transition_matrix) {
    const labels = data.labels || data.transition_matrix.map((_, i) => `S${i}`);
    const heatmapDiv = document.createElement('div');
    heatmapDiv.id = 'markovHeatmap';
    resultsDiv.appendChild(heatmapDiv);
    Plotly.newPlot('markovHeatmap', [{
        z: data.transition_matrix,
        x: labels,
        y: labels,
        type: 'heatmap',
        colorscale: 'Blues',
        text: data.transition_matrix.map(row => row.map(v => (v * 100).toFixed(1) + '%')),
        texttemplate: '%{text}',
        showscale: true
    }], {
        title: 'Transition Matrix Heatmap',
        height: 400,
        margin: { t: 50, l: 80, r: 20, b: 80 }
    });
}
```
