Create `static/js/healthScore.js` as a new file. Expose `window.HealthScore = { computeGrade, toggleDeepAnalysis }`. Also add `if (typeof module !== 'undefined' && module.exports) { module.exports = HealthScore; }` for testability.

**Module-level session state:**
```javascript
const _expandedTickers = new Set();  // cleared on new scrape via HealthScore.clearSession()
```
Expose `clearSession()` on the public API so stockScraper.js can call it before displayResults().

**`extractMetric(data, aliases)` helper (internal):**
- For each alias string, do a case-insensitive `key.toLowerCase().includes(alias.toLowerCase())` scan across all keys in `data`
- Skip values that are null, undefined, '', 'N/A', 'N/A%'
- Call `parseNumeric(val)` on the first match found
- Return null if no match

**`parseNumeric(val)` helper (internal):**
- If already a number, return it
- Strip leading '$', trailing '%', B/M/K multipliers: 'B'→×1e9, 'M'→×1e6, 'K'→×1e3 (only for non-ratio fields — but for ratios the scraper already returns plain numbers or percentage strings, so just strip '$' and '%' and parse float)
- Return null on NaN

**`scoreLiquidity(data)` → `{ letter, numericScore, rawValues, missing }`:**
- currentRatio = extractMetric(data, ['Current Ratio'])
- quickRatio = extractMetric(data, ['Quick Ratio'])
- If both null: return missing=true, letter=null, numericScore=null
- Score: start at 2 (C baseline)
  - currentRatio >= 2.0 → +1; currentRatio < 1.0 → -1
  - quickRatio >= 1.5 → +1; quickRatio < 0.5 → -1
  - Clamp to [0, 4]; map 4→A, 3→B, 2→C, 1→D, 0→F
- missing=true if either metric is null (partial data)

**`scoreLeverage(data)` → same shape:**
- debtEquity = extractMetric(data, ['Debt to Equity', 'D/E', 'Debt/Equity', 'Debt-to-Equity', 'Total Debt/Equity'])
- If null: missing=true, letter=null, numericScore=null
- Score: debtEquity < 0.5 → 4(A); < 1.0 → 3(B); < 2.0 → 2(C); < 3.0 → 1(D); >= 3.0 → 0(F)

**`scoreProfitability(data)` → same shape:**
- roe = extractMetric(data, ['ROE', 'Return on Equity', 'Return On Equity', 'ROE %'])
- profitMargin = extractMetric(data, ['Profit Margin', 'Net Margin', 'Net Profit Margin'])
- roa = extractMetric(data, ['ROA', 'Return on Assets'])
- If all null: missing=true, letter=null, numericScore=null
- Score: start at 2 (C baseline)
  - roe > 20 → +1; roe < 5 → -1   (values are percentage form, e.g. 25.3 not 0.253)
  - profitMargin > 20 → +1; profitMargin < 0 → -1
  - roa > 10 → +1; roa < 2 → -1
  - Clamp to [0, 4]; map to letter
- missing=true if any individual metric is null

**`scoreGrowth(data)` → same shape:**
- revenueGrowth = extractMetric(data, ['Revenue Growth', 'Sales Growth'])
- earningsGrowth = extractMetric(data, ['Earnings Growth', 'EPS Growth'])
- If both null: missing=true, letter=null, numericScore=null
- Score: start at 2 (C baseline)
  - revenueGrowth > 20 → +1; revenueGrowth < 0 → -1
  - earningsGrowth > 15 → +1; earningsGrowth < 0 → -1
  - Clamp to [0, 4]; map to letter

**Letter mapping:** `numericToLetter(n)` — 4→'A', 3→'B', 2→'C', 1→'D', 0→'F', null→null

**Grade colour class:** `gradeClass(letter)` — A→'badge-success', B→'badge-info', C→'badge-warning', D→'badge-warning', F→'badge-danger', null→'badge-warning'

**`buildExplanation(subScores)` helper:**
- Find sub-score with highest numericScore (positive driver) and lowest numericScore (negative driver), ignoring null scores
- Positive driver → label like "Strong {name}" (e.g. "Strong Profitability")
- Negative driver → label like "weak {name}" (e.g. "weak Leverage")
- If same score everywhere → return "Balanced financial profile."
- Return: "Strong {positive} offset by weak {negative}." or "Strong financial profile." if no negative driver below C

**`buildHTML(grade, gradeClass, subScores, explanation, ticker, isExpanded)` helper:**
Returns HTML string:
```html
<div class="deep-analysis-group" id="deep-analysis-group-{ticker}" style="border-top: 1px solid #e0e0e0; margin-top: 12px; padding-top: 10px;">
  <div class="deep-analysis-header"
       style="display:flex; justify-content:space-between; align-items:center; cursor:pointer; padding:6px 0;"
       onclick="HealthScore.toggleDeepAnalysis('{ticker}')">
    <span>🏥 Financial Health: <span class="badge {gradeClass}">{grade or 'N/A'}</span></span>
    <span class="deep-analysis-chevron" id="deep-analysis-chevron-{ticker}" style="transition:transform 0.3s;">{isExpanded ? '▲' : '▼'}</span>
  </div>
  <div class="deep-analysis-content" id="deep-analysis-content-{ticker}"
       style="display:{isExpanded ? 'block' : 'none'}; padding: 8px 0;">
    <div class="metric-group">
      {for each subScore: metric-item row}
    </div>
    <p style="margin:6px 0 0; font-size:13px; color:#555;">{explanation}</p>
  </div>
</div>
```
For each subScore row:
```html
<div class="metric-item">
  <span class="metric-label">{name}{missing ? ' ⚠' : ''}</span>
  <span class="metric-value">
    {letter ? `<span class="badge {gradeClass(letter)}">${letter}</span>` : '<span style="color:#999;">N/A</span>'}
    {rawValues summary string, e.g. ' — CR 2.1 / QR 1.6'}
  </span>
</div>
```
Raw values summary: show up to 2 non-null raw values as "KEY VAL" pairs separated by ' / '. Format floats to 1 decimal place. Skip null values.

**`computeGrade(data, ticker)` (public):**
- ticker defaults to 'unknown' if not provided (defensive)
- Call all four score functions
- Filter available = subScores where numericScore !== null
- overallNumeric = available.length > 0 ? avg(available.map(s => s.numericScore)) : null
- overallLetter = overallNumeric !== null ? numericToLetter(Math.round(overallNumeric)) : null
- explanation = buildExplanation(subScores)
- warnings = subScores.filter(s => s.missing).map(s => `${s.name} data incomplete`)
- isExpanded = _expandedTickers.has(ticker)
- html = buildHTML(overallLetter, gradeClass(overallLetter), subScores, explanation, ticker, isExpanded)
- Return: `{ grade: overallLetter, subScores, explanation, warnings, html }`

**`toggleDeepAnalysis(ticker)` (public):**
- Get content el: `document.getElementById('deep-analysis-content-' + ticker)`
- Get chevron el: `document.getElementById('deep-analysis-chevron-' + ticker)`
- If content hidden (display:none → display:block) — add to _expandedTickers, rotate chevron to '▲'
- If content visible (display:block → display:none) — remove from _expandedTickers, chevron back to '▼'

**`clearSession()` (public):** `_expandedTickers.clear()`
