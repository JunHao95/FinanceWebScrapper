## Cross-Phase Integration Gaps

1. **Markov sub-tab missing (MARKOV-01 to MARKOV-06):** The `/api/markov_chain` route supports 5 modes (steady_state, absorption, nstep, term_structure, mdp), all implemented and tested. But Phase 3 wired only the `nstep` mode inside the Credit Risk sub-tab for the heatmap. The dedicated Markov sub-tab (`stochContent_markov`) was never created.

2. **Vasicek UI path missing (RATE-02, RATE-03 partial):** `vasicek_yield_curve` was built in Phase 2, the `/api/interest_rate_model` route supports `model=vasicek`, but Phase 3 created only `stochContent_cir` without a model selector. No user can reach the Vasicek branch.
